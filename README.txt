REQUIREMENTS:
Python 3.7.4 64-bit
Feel free to modify the directories in the code to suit your computer, as well as the parameters/architecture of the network

HOW TO USE:
Step 1: Download this repository and extract it
Step 2: Set the paths for the four "pickle" files that contain the data for the training mutants, training labels, test mutants, and test labels
Step 3: Run the program, and you should receive a number of the accuracy of the network
